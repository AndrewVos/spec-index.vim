*spec-index.vim* Read and navigate your rspec/mocha/etc. tests in a quickfix
window.

Author:  Artem Avetisyan <https://github.com/artemave>
License: Same terms as Vim itself (see |license|)

USAGE

This plugin exports one function - ShowSpecIndex(). I map it like this:

    nnoremap <Leader>si :call ShowSpecIndex()<cr>
